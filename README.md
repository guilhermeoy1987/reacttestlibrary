# Requisitos
## Instalar as dependências
`npm install` ou `yarn install`

## Para executar os testes
`npm run test` ou `yarn test`

## Para executar a aplicação em localhost
`npm start` ou `yarn start`

# Testes Automatizados com React Testing Library - EBAC

Este repositório é focado na implementação de testes automatizados para garantir a qualidade e o funcionamento correto de componentes React.

## O que foi realizado:
* **Testes de Renderização:** Verificação da presença de elementos essenciais na interface.
* **Simulação de Usuário:** Testes de cliques e preenchimento de formulários para validar a interatividade.
* **Asserções com Jest:** Uso de expectativas para garantir que o comportamento do componente seja o esperado.
* **Cobertura de Código:** Criação de cenários de teste para evitar bugs durante o desenvolvimento.

## Tecnologias:
* React.js
* React Testing Library
* Jest
* JavaScript (ES6+)
* Git / GitHub
